#  API REST Empleados

- API RESTful para la gestión de empleados, usando arquitectura MVC, Spring JPA, Security, Lombok, Web.

# Tecnologías:

- Spring Web
- Spring JPA
- Spring Validation
- Spring lombok
- MySQL 
- Maven

# Características:

- Crud de empleados
- Validación de datos 
- Manejo global de excepciones
- DTO
- Paginación
